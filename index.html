🚀 How to publish on GitHub Pages and share with your boss
  • 1
    Go to github.com and create a free account (if you don't have one).
  • 2
    Click + → New repository. Name it siigo-converter. Set it Public and click Create repository.
  • 3
    Click "uploading an existing file" → drag this HTML file in → write a commit message → click Commit changes.
  • 4
    Go to Settings → Pages (left sidebar). Under Branch, select main → click Save. Wait ~2 minutes.
  • 5
    Your public URL will be: https://YOUR-USERNAME.github.io/siigo-converter/ — share this with your boss! Works on any device, no installation needed.
  • 6
    To update: just upload a new version of the HTML file to the same repository. GitHub Pages updates automatically.
